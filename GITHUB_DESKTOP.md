# GitHub Desktop ile Projeyi Açma

## Yöntem 1: Mevcut Repository'yi GitHub Desktop'ta Açma

1. **GitHub Desktop'ı açın**

2. **"File" > "Add Local Repository"** seçeneğine tıklayın

3. **Repository path'i seçin:**
   ```
   C:\Users\emirc\OneDrive\Masaüstü\disiplinlerarası
   ```

4. **"Add Repository"** butonuna tıklayın

5. GitHub Desktop projeyi otomatik olarak tanıyacak ve GitHub bağlantısını gösterecek

## Yöntem 2: GitHub'dan Clone Etme

1. **GitHub Desktop'ı açın**

2. **"File" > "Clone Repository"** seçeneğine tıklayın

3. **"URL" sekmesine gidin**

4. **Repository URL'ini girin:**
   ```
   https://github.com/emrcnclk/kiel-ai-full
   ```

5. **Local path seçin** (farklı bir yere clone etmek istiyorsanız)

6. **"Clone"** butonuna tıklayın

## Not

Eğer GitHub Desktop projeyi tanımazsa, proje klasöründe `.git` klasörünün olduğundan emin olun. Repository zaten başlatılmış durumda, bu yüzden GitHub Desktop otomatik olarak tanımalı.
